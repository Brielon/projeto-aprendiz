Tags essenciais para a utilização do HTML

Breve explicação

As tags em html servem para indicar ao navegador a forma na qual o site vai ser lido
Ou seja existem varias tags cada uma com uma funcionalidade especifica,sendo esse grupo dividido em dois: As tags semanticas e as tags nao semanticas

Tags Semanticas

Sao as tags que tem um significado ex: 'strong'
A tag 'strong' significa "forte" e faz com que a parte encapsulada por strong /strong fique em negrito

Tags nao semanticas

Ja as nao semanticas nao tem significado apenas alteram o texto ex: 'b'
A tag 'b' nao tem significado algun, contudo faz com que o texto encapsulado pelos mesmos fiquem em negrito
